## Installation and Execution
### Installation
* Download this repository
* Open the terminal (e.g., cmd or anaconda prompt)
* Execute the commands: 
```
$ cd the path that this repo was located on
$ pip install -r requirements.txt
```
### Execution
* Open the terminal (e.g., cmd or anaconda prompt)
* Execute the commands: 
```
$ cd the path that this repo was located on
$ python app.py
```
